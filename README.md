# Budget Tracker

## Description:
This is an app that allows you to keep track of your budget. You can add or subtract funds and track your spending to keep your budget in check! 

## Technologies used: 
* HTML
* CSS
* Javascript
* indexedDB
* MongoDB
* service-worker
